# BMW Car Catalog

A simple web application showcasing a catalog of BMW car models with filtering and details.

## Features

- Browse BMW car models by series (1, 2, 3, 4, 5, 7, 8 Series, X, Z, M)
- Filter cars by series, body style, and year
- View key specifications for each model
- Responsive design for desktop and mobile

## Getting Started

Open `index.html` in your browser to view the catalog.

No build tools or dependencies required — pure HTML, CSS, and JavaScript.
